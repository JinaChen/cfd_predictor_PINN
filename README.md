# Optimising Physics-Informed Neural Network Solvers for Turbulence Modelling

## Overview

This repository contains the code accompanying the study:

**Optimising Physics-Informed Neural Network Solvers for Turbulence Modelling: A Study on Solver Constraints Against a Data-Driven Approach**

The project investigates the use of Physics-Informed Neural Networks (PINNs) for turbulence-related flow modelling, with a particular focus on how different solver constraints influence optimisation behaviour, solution quality, and physical consistency. The work also compares constrained PINN-based solvers against a data-driven modelling approach.

The repository is intended to support reproducibility, experimentation, and further development of PINN-based solvers in computational fluid mechanics and turbulence modelling.

---

## Objectives

The main objectives of this repository are:

- implement PINN-based solvers for turbulence-related modelling tasks
- study the effect of solver constraints on optimisation performance
- compare physics-informed optimisation against a data-driven baseline
- provide a reproducible experimental framework for training, validation, and analysis
- support further research in scientific machine learning for fluid mechanics

---

## Features

- Physics-Informed Neural Network implementation for turbulence-related problems
- configurable loss components for PDE residuals, boundary conditions, and data terms
- comparison between constrained PINN solvers and data-driven models
- training, evaluation, and visualisation workflows
- support for reproducible experiments and result comparison
